## 🚀 Getting Started

### Clone Repository

```bash
git clone https://github.com/funnypar/login_page.git
```

### Go to Project Folder

```bash
cd login_page
```

### Install Dependencies

```bash
flutter pub get
```

### Run App

```bash
flutter run
```
